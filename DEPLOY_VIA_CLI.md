# Deploy via Render CLI (Alternative Method)

If the web interface button doesn't work, use CLI:

## Step 1: Install Render CLI
```bash
npm install -g render-cli
```

## Step 2: Login
```bash
render login
```
This will open a browser window for authentication.

## Step 3: Navigate to Backend
```bash
cd backend
```

## Step 4: Deploy
```bash
render deploy
```

The CLI will guide you through the setup interactively.
